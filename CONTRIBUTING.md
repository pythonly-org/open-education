# Contributing to Pythonly Open Education

Thank you for your interest in contributing!

## What you can contribute
- Educational content (Markdown, notebooks)
- Curriculum outlines and lesson plans
- Datasets and dataset explanations
- Typo fixes and clarifications

## How to contribute
1. Fork this repository
2. Create a feature branch
3. Submit a pull request with a clear description

## Guidelines
- Contributions should be educational and broadly reusable
- No proprietary or copyrighted material
- Keep content clear and beginner-friendly

## Code of Conduct
Be respectful and constructive. This is an educational community.
